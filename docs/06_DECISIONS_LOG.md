# سجلّ القرارات

## الطريقة العامة
الطريقة (ب): بناء محرّكات تفاعلية حقيقية لتبرير الاشتراك المدفوع. ترتيب البناء المتّفق: dragdrop → catch → trace → paint → balloon/connect → بديل خط اليد → توليد لانهائي.

## قرارات متّخذة
- التخزين ثنائي اللغة: jsonb موحّد {ar,fr} (لا أعمدة منفصلة).
- catch/balloon: نجاح بهدف معلن (الخيار ب)، balloon هدفه 12 للتكرار الكافي.
- trace: تغطية تراكمية 80% (أرحم لطفل صغير)، أرقام بـ centerline يدوي لا outline.
- connect: نقر لا سحب (السحب باللمس غير موثوق)، صحيح يختفي وخطأ يومض أحمر.
- generate: توليد محلّي خوارزمي لا AI (فوري، مجاني، offline).
- maze: مرفوضة وحُذفت.
- بديل خط اليد (نقطة 6): مغطّى بمحرّك trace، لا تعرّف خط يد فعلي في الكود.
- الدروس: محرّك lesson كبديل خفيف للفيديو (SVG/emoji + TTS) بدل فيديو حقيقي — لتفادي ثقل الفيديو وتكلفة الاستضافة وضعف الاتصال.

## قرارات مؤجّلة / مفتوحة
- مصدر المحتوى: مناهج الجزائر الرسمية (PDFs) لم تُرفع بعد.
- نظام XP/النقاط: المحرّكات تستدعي onComplete لكن لا تمنح XP فعليًا بعد.
- الصوت: TTS المتصفّح حاليًا. مخطّط: ملفات mp3 من speechma.com (مجاني، ترخيص تجاري، لا API → توليد يدوي ورفع). يصلح للدروس الثابتة لا التوليد اللانهائي. صوت Amina (أنثى عربية) مرشّح، ضبط Pitch قيد التجربة. التخزين المرجّح: داخل التطبيق public/audio/ (offline).
- تعميم محرّك lesson: المعاينة والمحتوى مكتوبان يدويًا للجمع؛ يلزم اشتقاقهما من data.
