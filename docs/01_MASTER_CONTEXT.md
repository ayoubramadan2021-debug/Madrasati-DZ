Project Name

تعليم DZ (Taalim DZ)

Vision

منصة تعليمية جزائرية تفاعلية للابتدائي (1–5) تعتمد على التعلم الممتع، التقدم بالممارسة، والذكاء الاصطناعي المساعد.

Educational Structure

Subject
→ World
→ Lesson
→ Content (شرح + تمارين)

كل World يحتوي اختبار بوابة لفتح العالم التالي.

Business Model

- فتح السنة الدراسية بدل الاشتراك التقليدي
- السعر المستهدف: 2500 دج لكل سنة
- كل سنة تُفتح بشكل مستقل
- السنوات السابقة تبقى مفتوحة دائمًا للمراجعة
- Freemium محدود + محتوى Premium

Gamification

- Worlds قابلة للفتح
- XP / Stars / Progress
- اختبارات بوابة
- تحديات
- تكريم وطني/ولائي

AI System

Rules
