## 1. الفلسفة (لا تنسَها)
═══════════════════════════════════════════════
- **الدرس** (ذهبي/دافئ): خليل حاضر، استكشاف حرّ، بلا تقييم، بلا نقاط. الطفل يشاهد ويسمع.
- **التمرين** (بنفسجي/أخضر): تقييم بنقاط، خليل في البداية/النهاية فقط. خطأ = أحمر + إعادة نفس النشاط حتى الصواب، بلا عقاب، بلا تخطّي.
- كل درس = **6 مشاهد + 3 تمارين × 5 أسئلة**.

## 5. مصائد تقنية (تعلّمناها — لا تكرّرها)
═══════════════════════════════════════════════
- **EmojiIcon لا يُحدّث بين الأسئلة**: key فريد على span = `${itemIdx}-${i}-${e}`
- **تبديل نفس المحرّك بين مراحل**: key="exN" على المكوّن في الصفحة
- **سباق الصوت**: استخدم useKaraoke المشترك (genRef token). استدعِ karaoke.stop() قبل صوت "أحسنت".
- **edge-tts**: WordBoundary إلزامي. التاء المربوطة بلا حركة تُبتلع.
- **sed على الموبايل**: تجنّب # و| كفاصل مع نصوص فيها رموز؛ استخدم @. \n متعدد الأسطر يفشل. تحقّق بـgrep بعده (sed يكذب بالنجاح).
- **Python str_replace صامت الفشل** → full rewrite (cat<<EOF) أو read+replace+write.
- **dev cache**: لو لم يظهر تغيير → أوقف dev، أعد npm run dev، تبويب خفي.
- **Pollinations مدفوع (402)** → ChatGPT يدوي + ضغط.
- **تحقّق دائماً**: الأوامر قد لا تُنفَّذ. grep -c بعد كل تعديل قبل البناء.
